================================================================================
                           HÄUFIGE PROBLEME
================================================================================

1. "You're accessing the development server over HTTPS..."
   → Die Person hat https:// statt http:// eingegeben.
   → Lösung: Im Browser "http://" statt "https://" verwenden.

2. Seite lädt nicht / Verbindung verweigert
   → Windows Firewall blockiert Port 8000.
   → Oder falsche IP-Adresse.

3. "Bad Request" oder seltsame Fehler
   → Fast immer HTTPS statt HTTP.
